# Devclub

Simple PHP CRUD example project for DevClub.

## Files
- `index.php` — List records
- `add.php` — Add a record
- `edit.php` — Edit a record
- `delete.php` — Delete a record
- `db.php` — Database connection
- `devclub.sql` — DB schema

## How to run
1. Put files in `htdocs/DevClub` (XAMPP)
2. Import `devclub.sql`
3. Open http://localhost/DevClub/index.php
